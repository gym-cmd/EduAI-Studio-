# Google Cloud Hackathon 19

## Overview

We are building a curriculum builder. You tell us what you want to learn, we make roadmap for you to reach your goals, create learning content and educational quizes. 

## Spec

All feature development starts with a spec. Before writing code, a spec must exist and be approved.

- Specs live in [`specs/`](specs/)
- Each spec follows the template in [`specs/_template.md`](specs/_template.md)
- A spec must include: problem statement, proposed solution, acceptance criteria, and out-of-scope items

I want to build a web app that would be a personal tutor for learning software development. it would have user's profiles and saved info abouit their intersts, educational level etc. then you could start a new chat, the tutor would ask you what you want to learn, get extra details and then move on to building the curriculum and expand on the next steps in it. then it would combine the sources it found, present the links + some overview and also create a quiz. after doing the quiz it will show the score and tell you what to revise or offer to generate new learning step
